# homebrew-tap

Homebrew tap for [moond4rk](https://github.com/moond4rk) projects.

## Installation

```bash
brew tap moond4rk/tap
```

## Available Packages

| Package | Description |
|---------|-------------|
| [ccstatus](https://github.com/moonD4rk/ccstatus) | Customizable status line formatter for Claude Code CLI |
| [keychainbreaker](https://github.com/moonD4rk/keychainbreaker) | Extract credentials from macOS Keychain files |
| [things3](https://github.com/moonD4rk/things3) | CLI for Things 3 task manager on macOS |

### ccstatus

```bash
brew install moond4rk/tap/ccstatus
```

### keychainbreaker

```bash
brew install moond4rk/tap/keychainbreaker
```

### things3

```bash
brew install moond4rk/tap/things3
```
